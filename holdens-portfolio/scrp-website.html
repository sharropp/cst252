SANTA CRUZ ROLLER PALLADIUM

Programs: Visual Studio Code, Figma, Replit

This is a hypothetical website redesign for a local business in Santa Cruz, “Santa Cruz Roller Palladium”. This project expanded upon my learning from my “London Bridge Pub” project and featured fancy CSS curve shapes, CSS animation (image galleries), and responsive page design for wider monitors (content stops scaling up to a certain size). I feel I also took more time to stylize the color, typography, imagery, and overall visual design and layout of this site compared to prior web projects, which felt more modular.
