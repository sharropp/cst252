LONDON BRIDGE PUB

Programs: Visual Studio Code, Figma, Replit

This is a hypothetical website redesign for a local restaurant in Monterey, “London Bridge Pub”. This was the first time I had recreated an entire website from the ground up, creating sketches and a Figma prototype for each page (after researching competing sites), and even creating a new logo. This project was also my first experience with flexbox, responsive design (media queries), and front-end forms.
